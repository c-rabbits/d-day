# 레벨 테이블 (1~100 전체, 누적 XP)

- **최대 레벨**: 100  
- **XP 상한**: 999,999 (100레벨 달성 시)  
- **곡선**: 초반 빠르게 레벨 업, 후반으로 갈수록 느리게 (지수 1.8)  
- **공식**: 누적 XP(L) = `round(999999 × ((L-1)/99)^1.8)` (L=1이면 0)
